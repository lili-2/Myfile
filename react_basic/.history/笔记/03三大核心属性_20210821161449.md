1. state

    1. 理解
        - state是组件中最重要的属性.值是对象(可以包含多个key-value组合) 
        - 组件被称为状态机 通过更新组件中的state来更新对应的页面显示(重新渲染组件render)
    2. 注意
        1) 组件中的render方法中的this为组件的实例对象
        2) 组件自定义的方法中this为undefined(作为事件回调使用),解决
            a:强制绑定this,通过函数对象的bind()
            b:箭头函数
        3) 状态函数 不能直接修改或更新
